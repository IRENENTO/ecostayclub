# EcoStay Club

A sustainable community platform for environmental awareness and action.

## Features

- **Community Dashboard**: Track environmental impact with trees planted, carbon saved, and member statistics
- **Admin Panel**: Manage users, content, and site data
- **Progressive Web App**: Installable with offline capabilities
- **Responsive Design**: Works on all devices
